# Video Editor - CapCut Style 🎬

A modern, browser-based video editing platform built with React, Vite, and Tailwind CSS.

## Features ✨

### Phase 1 (Current)
- ✅ Video Upload (Drag & Drop)
- ✅ Video Preview with Real-time Filters
- ✅ Playback Controls (Play/Pause)
- ✅ Speed Control (0.5x - 2x)
- ✅ Video Effects:
  - Brightness Adjustment
  - Contrast Adjustment
  - Saturation Control
- ✅ Timeline-based Trimming
- ✅ Time Display & Navigation

### Phase 2 (Coming Soon)
- Audio Upload & Mixing
- Text & Stickers
- Transitions
- Advanced Filters
- Video Export (MP4)
- Multiple Clips Support
- Layer Management

## Tech Stack 🛠️

- **React 18** - UI Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Lucide Icons** - Icons

## Installation & Setup 🚀

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Steps

```bash
# Clone the repository
git clone https://github.com/aashumalik784/video-editor.git
cd video-editor

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The app will open at `http://localhost:3000`

## Usage 📹

1. **Upload Video**: Drag & drop or click to upload a video file
2. **Edit**: Use the controls and effects panel to edit your video
3. **Trim**: Adjust start and end times in the timeline
4. **Preview**: Play the video to preview changes
5. **Export**: Download your edited video (coming soon)
